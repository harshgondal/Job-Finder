# Job Application Finder - Agent-Based System

## Overview

Job Application Finder combines AI agents to read resumes, refine preferences, normalize live job feeds, and explain why each role fits so candidates can act quickly.

## Key Features

- Automated resume parsing with preference refinement
- Normalized job listings scored with transparent reasoning
- Company research briefs with interview-ready talking points

## Tech Stack

- Backend: Node.js, Express, Gemini AI, RapidAPI JSearch, MongoDB, Redis
- Frontend: React 18, TypeScript, Vite
